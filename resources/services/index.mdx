All services are open source and self-hostable.

## Services

- [ActivePieces](./activepieces) - Open source no-code business automation.
- [Appsmith](./appsmith) - A low-code application platform for building internal tools.
- [Appwrite](./appwrite) - A backend-as-a-service platform that simplifies the web & mobile app development.
- [Authentik](./authentik) - An open-source Identity Provider, focused on flexibility and versatility.
- [Baby Buddy](./babybuddy) - It helps parents track their baby's daily activities, growth, and health with ease.
- [Budge](./budge) - A budgeting personal finance app.
- [Change Detection](./changedetection) - Website change detection monitor and notifications.
- [ClassicPress](./classicpress) - A business-focused CMS with a strong community.
- [Code Server](./code-server) - Run VS Code on any machine anywhere and access it in the browser.
- [Dashboard](./dashboard) - A simple dashboard for your server.
- [Directus](./directus) - An open-source headless CMS and API for custom databases.
- [Docker Registry](../../knowledge-base/docker/registry) - A Docker registry to store and manage your Docker images.
- [DocuWiki](./dokuwiki) - A simple to use and highly versatile Open Source wiki software that doesn't require a database.
- [Duplicati](./duplicati) - A free backup client that securely stores encrypted, incremental, compressed backups on cloud storage services and remote file servers.
- [Emby](./emby) - A media server to organize, play, and stream audio and video to a variety of devices.
- [Emby Stat](./emby-stat) - A simple and easy-to-use Emby statistics dashboard.
- [Fider](./fider) - An open platform to collect and organize customer feedback.
- [Filebrowser](./filebrowser) - A file manager for the web.
- [Firefly III](./firefly-iii) - A personal finances manager.
- [Formbricks](./formbricks) - A form builder for static sites.
- [Ghost](./ghost) - A professional publishing platform.
- [Gitea](./gitea) - A painless self-hosted Git service.
- [GlitchTip](./glitchtip) - An open-source error tracking tool.
- [Grafana](./grafana) - The open platform for beautiful analytics and monitoring.
- [Grocy](./grocy) - A self-hosted groceries & household management solution for your home.
- [Heimdall](./heimdall) - An elegant solution to organize all your web applications.
- [Jellyfin](./jellyfin) - The Free Software Media System.
- [Kuzzle](./kuzzle) - A powerful backend that enables you to build modern apps faster.
- [Logto](./logto) - Logto is an Auth0 alternative designed for modern apps and SaaS products.
- [MeiliSearch](./meilisearch) - A powerful, fast, open-source, easy to use, and deploy search engine.
- [Metabase](./metabase) - The simplest, fastest way to share data and analytics inside your company.
- [Metube](./metube) - A self-hosted video sharing platform.
- [Plausible Analytics](./plausible) - Simple and privacy-friendly alternative to Google Analytics.
- [MinIO](./minio) - A high-performance, distributed object storage system.
